How to use Drop Jumbo

1. Include Drop on your site.

<link rel="stylesheet" href="drop-jumbo.css">
<script src="drop.js"></script>

If you want multiple columns of content within your dropdown and don't already have a grid system, also include kraken-grid.css.

2. Add the markup to your HTML.

This example uses the Kraken grid system. Feel free to replace it with your own.

        <a href="#">Dropdown 1</a>

3. Add a feature test.

Drop uses modern JavaScript API's that aren't supported by older browsers. Add this short snippet of code in the <head> of your HTML document to feature test:

<!-- Feature Test -->
<script>
    if ( 'querySelector' in document && 'addEventListener' in window && Array.prototype.forEach ) {
        document.documentElement.className += 'js';
    }
</script>

For browsers that support modern JavaScript API's, this snippet adds a .js class to the <html> element, which activates dropdown-specific styling elements. Older and less capable browsers get a basic link.
